# Voice In (ボイス・イン)

**あなたの声を、あらゆる場所でテキストに。**

Voice In は、PC上のあらゆるアプリケーションで使える、AI搭載の次世代音声入力ツールです。
キーを押し続けて話すだけ。最新のAIがあなたの言葉を理解し、句読点の追加や文法修正を行い、完璧な文章として入力します。

[English](docs/README_en.md) | [한국어](docs/README_ko.md) | [中文](docs/README_zh.md) | [Français](docs/README_fr.md)

## ✨ なぜ Voice In なのか？

- **どこでも入力**: メモ帳、ブラウザ、Slack、コーディングエディタ... アクティブなウィンドウならどこでも入力可能です。
- **ただの文字起こしではありません**: "えーっと" などの不要な言葉を取り除き、文脈に合わせて「です・ます」調や箇条書きなどに自動で整形します。
- **選べるAI**: 最新のクラウドAI（Groq, Google Gemini）による超高速・高精度な認識に加え、プライバシー重視のローカル処理（Local Whisper）も選択可能です。
- **クロスプラットフォーム**: Linux をメインに、Windows や macOS でも動作します。

## 🚀 はじめ方

### 1. ダウンロード (Download)
**Windows / macOS / Linux 用の実行ファイル（.exe / .app / .bin）を配布しています。**

右側の [Releases](https://github.com/watawatan1984/voice-inn-linux/releases) ページから、お使いのOSに合った最新のファイルをダウンロードしてください。
インストール不要で、ダウンロードしてそのまま起動できます。

### 2. インストールと起動
ダウンロードしたファイルを実行するか、以下のコマンド（上級者向け）でセットアップします。

> **初心者の方へ**: 多くのLinux環境では、ダウンロードしたファイルを実行権限付きで起動するだけで使えます。

### 3. 設定 (Setup Wizard)
初回起動時にセットアップウィザードが開きます。
1. **AIプロバイダ**: 高速な `Groq`、高精度な `Gemini`、またはオフラインの `Local` から選びます。
2. **APIキー**: クラウドAIを使用する場合は、無料のAPIキーを取得して入力してください。
3. **マイク選択**: 使用するマイクを選び、テストします。

## 🎙️ 使い方

使い方はとてもシンプルです。

1. 文字を入力したい場所（テキストボックスなど）をクリックします。
2. **`Left Alt` キー（または設定したキー）を押し続けます**。
3. マイクに向かって話します。「マイクアイコン」が画面に表示されます。
4. 話し終わったらキーを離します。
5. AIが文章を整え、自動的に入力されます ✨

## ⚙️ 詳細設定とカスタマイズ

タスクトレイのアイコンを右クリックして `Settings` を開くと、より詳細な設定が可能です。

### 1. General (一般設定)

基本的な動作環境を設定します。

- **AI Provider (AIプロバイダ)**
  - `Groq`: 超高速。インターネット接続が必要。
  - `Gemini`: 高精度で長文に強い。インターネット接続が必要。
  - `Local`: PC内部で処理して完結します。プライバシー重視・オフライン環境向け。（初回のみモデルダウンロードが発生）
- **Gemini Model / Groq API Key**: 選択したプロバイダに応じた設定値を入力します。
- **Input Device (入力デバイス)**: 使用するマイクを選択します。
- **Input Gain (入力ゲイン)**: マイクの入力音量を調整します。声が小さい場合は数値を上げてください。
- **Hold Key (録音キー)**: どのキーを押している間に録音するかを変更できます（デフォルト: `Left Alt`）。
- **Max Recording (最大録音時間)**: 一度の入力の最大秒数です。これを超えると自動で文字起こしが始まります。
- **Min Valid Duration (最小有効録音時間)**: これより短い音声は無視します。誤ってキーに触れた場合などの誤作動を防ぎます。
- **Auto Paste (自動貼り付け)**: ONにすると、文字起こし完了後に自動でテキストを入力します。OFFの場合はクリップボードへのコピーのみ行います。
- **Paste Delay (貼り付け遅延)**: テキスト入力までの待機時間(ms)です。入力先アプリへのフォーカス切替が間に合わない場合に増やしてください。
- **Language (表示言語)**: アプリのメニュー言語を切り替えます。

### 2. Prompts (プロンプト)

AIへの指示をカスタマイズして、文字起こしのスタイルを変更できます。

- **Groq Whisper Prompt**: 音声認識のヒントを与えます（例：「医学用語の書き起こしです」）。
- **Refine System Prompt**: 文字起こし後の整形ルールです。「関西弁にして」「英語に翻訳して」「要約して」といった指示をここに書きます。

### 3. Dictionary (辞書)

よく使う単語や、AIが間違えやすい固有名詞を登録します。

- **From (置換前)**: AIが出力しがちな誤字（例: "ボイスイン"）
- **To (置換後)**: 正しい表記（例: "Voice In"）
- 文字起こし結果に対して、このルールの通りに強制的に置換を行います。

### 4. Local Whisper

`Local` プロバイダを選択した場合の詳細設定です。

- **Model Size**: モデルの大きさです。`large-v3` が最も高精度ですが、メモリを消費し遅くなります。`base` や `small` は高速です。
- **Device**: `cuda` (NVIDIA GPU) または `cpu` を選択します。GPUがない場合は `cpu` にしてください。
- **Compute Type**: 計算精度です。`float16` が基本ですが、メモリ不足の場合は `int8` を試してください。
